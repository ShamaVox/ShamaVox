# GoalCoach AI (Expo + TypeScript)

A tiny side‑hustle app you can ship on iOS/Android: weekly goals + mindful micro‑check‑ins.  
No backend required; uses AsyncStorage. Brand‑neutral and ready to extend with LLMs.

## Features
- Weekly goal setting
- CBT‑style reflection modal
- Local persistence (Zustand + AsyncStorage)
- Insights screen with simple metrics

## Run Locally
```bash
npm i -g expo-cli
npm i
npm run start   # press i for iOS, a for Android
```

## Monetization Ideas
- Freemium: 3 goals/week free → Pro $2.99/mo for unlimited + export
- Add LLM‑powered “coach replies” as a Pro feature
